# India GPT Backend

## Setup
1. Copy `.env.example` to `.env` and fill values:
   - `ALLOWED_ORIGIN` set to your Netlify site URL
   - `BING_SEARCH_KEY` from Azure Cognitive Services (Web Search)
   - `OPENAI_API_KEY` (or Azure OpenAI key)
2. Install: `npm install`
3. Run: `npm start`

## Deploy
- Render / Railway / Fly.io
- Expose `PORT`
- Add environment variables in the service dashboard
